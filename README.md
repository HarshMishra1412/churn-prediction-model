# 🔮 Customer Churn Prediction Model

## 📌 Project Overview
An end-to-end Machine Learning model to predict 
customer churn for a Telecom company using XGBoost!

## 📊 Results
| Metric   | Score |
|----------|-------|
| Recall   | 92%   |
| ROC-AUC  | 0.76  |
| Accuracy | 69%   |

## 🛠️ Technologies Used
- Python
- XGBoost
- Scikit-learn
- SMOTE (Imbalanced-learn)
- Pandas, NumPy
- Seaborn, Matplotlib

## 🔄 Project Steps
1. Data Cleaning & Preprocessing
2. Label Encoding
3. SMOTE for Class Imbalance
4. Model Training (XGBoost)
5. GridSearchCV Optimization
6. Threshold Tuning (0.35)
7. Feature Importance Analysis

## 📈 Key Findings
- Contract type is the most important churn factor
- Month-to-month customers churn the most
- Best recall achieved at threshold 0.35
